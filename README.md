# ⚡ Facebook Automation Extension

**Chrome Extension untuk otomatisasi aktivitas Facebook**
Auto reply · Auto follow · Posting · Manajemen Grup · dan masih banyak lagi

*Dikembangkan oleh **Khamdihi Dev** — fokus pada automation yang aman, ringan, dan powerful.*

---

</div>

## 🔷 Fitur Utama

### 📌 Grab UID
Ambil UID pengguna secara otomatis dari dua sumber:
- 🎯 **Teman Publik** — dump UID dari profil teman
- 🎯 **Saran Teman** — dump dari halaman *People You May Know*

---

### 📌 Add Teman
Tambahkan teman secara otomatis dengan kontrol penuh:
- 💡 Input **list UID** sekaligus
- 💡 Sistem **delay antar request** untuk menghindari limit Facebook

---

### 📌 Posting Foto
Post gambar ke timeline dengan mudah:
- 💡 Upload file foto dari perangkat
- 💡 Input **caption custom** sesuai kebutuhan

---

### 📌 Posting Reels *(Coming Soon)*
> ⏳ Fitur ini masih dalam tahap pengembangan dan akan hadir di versi berikutnya.

---

### 📌 Dump Group by Keyword
Temukan grup Facebook berdasarkan topik:
- 💡 Input **keyword** pencarian
- 💡 Ambil daftar group yang relevan secara otomatis

---

### 📌 Kirim Pesan (Auto Chat)
Kirim pesan massal dengan fleksibel:
- 🎯 **Ambil daftar teman otomatis** dari akun
- 🎯 Atau **input UID manual** sesuai target
- 🎯 Support **random message** untuk variasi pesan:
  ```
  halo|hai|apa kabar?|selamat pagi
  ```
  > 💡 Pesan akan dipilih secara acak untuk setiap pengiriman

---

### 📌 Join Group
Masuk ke banyak group sekaligus:
- 💡 Input **list UID group**
- 💡 Auto join semua group dalam daftar

---

### 📌 Posting ke Group
Posting konten ke grup dengan berbagai mode:

| Mode | Keterangan |
|------|-----------|
| 🎯 **Anonymous Post** | Posting tanpa menampilkan identitas |
| 🎯 **Normal Post** | Posting standar dengan akun aktif |
| 🎯 **Random Foto** | Foto dipilih acak dari daftar yang disiapkan |
| 🎯 **Random Caption** | Caption dipilih acak, dipisah dengan `\|` |

---

### 📌 Keluar Group
Tinggalkan banyak grup sekaligus:
- 💡 Input **list UID group**
- 💡 Auto leave semua group dalam daftar

## 🔷 Cara Penggunaan

```
1. Buka Facebook di browser Chrome
2. Aktifkan extension dari toolbar
3. Pilih menu sesuai kebutuhan
4. Masukkan data yang diperlukan (UID, caption, foto, dll)
5. Klik ▶ Start
```

> 💡 **Tips:** Selalu gunakan fitur **delay** untuk menghindari deteksi spam oleh sistem Facebook.
